Tribute to Mario SpeedRunnning

History (120 star)

  • 1996: On June 23rd, Mario 64 was released.
  • 2004: 8 years after the release the first recorded speedrun with a time of 2:47:57 is set by spiderman88mil
  • 2006: 2 years after the first run a 2:09:40 is set by Dragorn
  • 2010: 4 more years of inactivity until Batora hit a time of 1:52:02
  • 2011-2013: The run was shortened many times, but ends with Siglemic getting a 1:44:01
  • 2014: Over a year later Siglemic shortened his own time to a 1:43:53
  • 2019: ikori sets a new record with a 1:40:44
  • 2020: In February cheese sets a new record with a 1:38:54
  • 2020: 4 months later Simply beats his friend cheese with a 1:38:28

Context

Super Mario 64 is the first 3D platforming game in the Mario Series created by Nintendo. To get through the game one must collect a certain number of "power stars" to advace through the castle. This page will cover many different categories of runs with the difference being the number of "power stars" collected since you can use glitches in the game to get through certain parts of the castle.

120 star

This is every single star in the game. These runs usually are 1 hour 45 minutes long.This is the hardest run to do since it is the longest. The current world record is 1:38:28 by the runner simply.

WR RUN

70 star

This is the minimum amount of stars needed to beat the game without any glitches. These runs are usually 50 minutes and are a good middle ground for runners. The current world record is 46:59 by Dwhatever.

WR RUN

16 star

This is a good beginner run for the game. It requires the use of a glitch in the game called a BLJ or backwards long jump. These runs are usually 20 minutes. The current world record is 14:59 by akki

WR RUN

Tool Assisted Speedruns

Tool Assisted Speedruns or also know as TAS runs are some of the most optimized runs in any game due to the fact that coding scripts are used to control the game. In mario there are frame perfect inputs done in these runs that are humanly impossible. I would suggest just watching the TAS world record for mario 64 as it is impressively optimized
